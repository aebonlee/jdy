# 직업미래연구소 (Job Future Institute)

**정동엽 직업학박사의 공식 웹사이트**

> 미래사회와 직업세계 연구 | 커리어 경력설계 | 전직지원 | 창업·창직 컨설팅

## Live

**https://jdy.dreamitbiz.com/**

## Features

- **반응형 디자인** — 데스크톱, 태블릿, 모바일 완벽 대응
- **한/영 언어 전환** — 네비바 EN/KO 토글 버튼으로 즉시 전환 (약 120개 키 번역)
- **다크/라이트 테마** — 해/달 아이콘 토글, localStorage 저장으로 재방문 시 유지
- **스크롤 애니메이션** — Intersection Observer 기반 fade-in/fade-up 효과
- **카운터 애니메이션** — 통계 수치 카운트업 (ease-out cubic)
- **카카오톡 오픈채팅** — QR코드 스캔 또는 버튼 클릭으로 바로 상담
- **OG 미리보기** — 카카오톡·트위터 공유 시 1200x630 전용 이미지 표시

## Tech Stack

| 항목 | 기술 |
|------|------|
| HTML | Semantic HTML5 |
| CSS | CSS Custom Properties, Glassmorphism, Grid/Flexbox |
| JS | Vanilla JavaScript (no framework) |
| Font | Pretendard (KR) + Outfit (EN) |
| Deploy | GitHub Pages + GitHub Actions |
| Domain | jdy.dreamitbiz.com (CNAME) |
